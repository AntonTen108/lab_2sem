    Лабораторная работа 1 — Сумма двух чисел (Two Sum)¶

    Цель¶

    Реализовать функцию, которая находит в списке два числа, сумма которых равна заданному значению target, и возвращает их индексы.

    Задание¶

    Дан список целых чисел nums и число target.
    Найти индексы двух элементов, сумма которых равна target.
    Предполагается, что решение всегда существует, и один элемент не может использоваться дважды.

    Реализация¶

    def sumof2(nums, target):   # создаем функцию
    n = len(nums)   # вводим переменную n со значением количества элементов списка nums
    for i in range(n - 1):  # вводим переменную i от 0 до предпоследнего элемента nums
        for k in range(i + 1, n):   # вводим переменную k от 1 до последнего элемента nums
            if target - nums[i] == nums[k]:     # условие: разность target и nums[i] равна nums[k]
                return [i, k]
    return None

result = sumof2([1,2,3,4,5,6,7,8,9], 4)
print(result)

    Описание алгоритма¶

    Используется метод перебора всех пар (brute force):

    1. Внешний цикл перебирает каждый элемент nums[i].
    2. Внутренний цикл перебирает все следующие элементы nums[k].
    3. Проверяется условие: target - nums[i] == nums[k].
    4. Если условие выполнено — возвращаются индексы [i, k].

    Сложность алгоритма: O(n²) по времени, O(1) по памяти.

    Пример работы¶

    Вход: nums = [1,2,3,4,5,6,7,8,9], target = 4

    Ищем пару, сумма которой равна 4: - nums[0] = 1, nums[2] = 3 → 1 + 3 = 4 ✓

    Результат: [0, 2]

    Выводы¶

    В ходе работы освоены вложенные циклы и работа со списками в Python.
    Алгоритм перебора прост в реализации, однако при больших размерах списка его производительность снижается. Для оптимизации можно использовать хэш-таблицу (словарь), что снизит сложность до O(n).
