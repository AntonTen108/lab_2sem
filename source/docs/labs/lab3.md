# Лабораторная работа 3 — Генерация бинарного дерева

## Цель

Реализовать рекурсивную генерацию бинарного дерева заданной высоты с представлением в виде вложенного словаря Python.

---

## Задание

Написать функцию, которая строит бинарное дерево заданной высоты. Левый дочерний узел вычисляется как `root * 2`, правый — как `root + 3`. Дерево представляется в виде вложенного словаря.

---

## Реализация

```python
from pprint import pprint

def gen_bin_tree(root, height):
    if height == 0:
        return {root: {}}

    left_root = root * 2
    right_root = root + 3

    return {
        root: {
            **gen_bin_tree(left_root, height - 1),
            **gen_bin_tree(right_root, height - 1),
        }
    }


res = gen_bin_tree(1, 3)
pprint(res)
```

---

## Описание алгоритма

Функция `gen_bin_tree` работает рекурсивно:

1. **База рекурсии**: если `height == 0`, возвращается словарь `{root: {}}` — лист дерева.
2. **Рекурсивный шаг**: вычисляются два дочерних узла и для каждого рекурсивно строится поддерево.
3. Результаты объединяются оператором `**` (распаковка словарей).

Правила построения узлов:

- Левый потомок: `root * 2`
- Правый потомок: `root + 3`

## Пример вывода

Дерево высотой 3 с корнем 1:

```
{1: {2: {4: {8: {}, 7: {}},
          5: {10: {}, 8: {}}},
     4: {8: {}, 7: {}}}}
```
