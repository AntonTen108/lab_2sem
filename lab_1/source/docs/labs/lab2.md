# Лабораторная работа 2 — Бинарный поиск

## Цель

Реализовать алгоритм бинарного поиска элемента в отсортированном списке с подсчётом количества шагов.

---

## Задание

Пользователь вводит искомое число и диапазон поиска. Программа должна найти число в списке методом бинарного поиска и вывести результат и количество затраченных попыток.

---

## Реализация

```python
def guess_number(target, lst, type='bin'):      # type='bin' - метод поиска (бинарный)
    counter = 0
    if target < lst[0] or target > lst[-1]:
        return None, counter

    elif type == "bin":
        start, end = 0, len(lst) - 1
        while start <= end:
            counter += 1
            middle = (start + end) // 2

            if lst[middle] == target:
                return lst[middle], counter
            elif lst[middle] < target:
                start = middle + 1
            else:
                end = middle - 1
        return None, counter

if __name__ == '__main__':
    target = int(input("Введите искомое число "))
    start = int(input("Введите начало диапазона "))
    end = int(input("Введите конец диапазона "))
    lst = list(range(start, end + 1))
    result, steps = guess_number(target, lst, type='bin')
    if result is None:
        print("Загаданное число не входит в заданный вами диапазон")
    else:
        print("Загаданное вами число =", result)
        print("Количество попыток =", steps)
```

---

## Описание алгоритма

**Бинарный поиск** работает по принципу деления пополам:

1. Берётся средний элемент списка.
2. Если он равен искомому — возвращаем результат.
3. Если искомое больше среднего — ищем в правой половине.
4. Если меньше — ищем в левой половине.
5. Повторяем до нахождения или исчерпания диапазона.

Сложность: **O(log n)** — значительно эффективнее линейного поиска.
