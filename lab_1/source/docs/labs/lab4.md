# Лабораторная работа 4 — Факториал: рекурсия и итерация

## Цель

Реализовать вычисление факториала двумя способами — рекурсивным и итеративным — и сравнить подходы.

---

## Задание

Написать две функции для вычисления факториала числа `n`:

1. `fact_recursive` — с использованием рекурсии.
2. `fact_iterative` — с использованием цикла.

Обе функции должны корректно обрабатывать граничные случаи: `n = 0`, `n = 1`, `n < 0`.

---

## Реализация

```python
def fact_recursive(n):
    if n < 0:
        return None
    if n == 0:
        return 1
    if n == 1:
        return 1
    return fact_recursive(n - 1) * n

def fact_iterative(n):
    if n < 0:
        return None
    if n == 0:
        return 1
    fact = 1
    for i in range(1, n + 1):
        fact *= i
    return fact

print(fact_recursive(99))
```

---

## Описание алгоритмов

### Рекурсивный подход

Функция вызывает саму себя с уменьшенным аргументом:

- `fact(n) = n * fact(n-1)`
- База рекурсии: `fact(0) = 1`, `fact(1) = 1`

### Итеративный подход

Используется цикл `for`, который последовательно перемножает числа от 1 до `n`.

### Сравнение

| Критерий | Рекурсия | Итерация |
|----------|----------|----------|
| Читаемость | Высокая | Средняя |
| Использование стека | O(n) | O(1) |
| Риск переполнения стека | Есть | Нет |
| Скорость | Немного медленнее | Быстрее |

## Выводы

Реализованы два подхода к вычислению факториала. Рекурсивный вариант более нагляден и математически точен, однако при больших значениях `n` итеративный подход предпочтительнее из-за экономии памяти стека. Python поддерживает длинную арифметику, что позволяет вычислить факториал очень больших чисел без ошибок переполнения.
